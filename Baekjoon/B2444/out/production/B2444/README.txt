백준 2444번 (별찍기 -7)

문제: 예제를 보고 규칙을 유추한 뒤에 별을 찍어 보세요.
입력: 첫째 줄에 N(1 ≤ N ≤ 100)이 주어진다.
출력: 첫째 줄부터 2×N-1번째 줄까지 차례대로 별을 출력한다.

예제입력: 5     예제출력:
                                *
                               ***
                              *****
                             *******
                            *********
                             *******
                              *****
                               ***
                                *
빈칸
n-1개부터 시작해서 0이 됐다가 다시 n-1개가 됨

별
5번째 줄을 기준으로 늘어났다가 줄어듦

흐름
1. Bufferedreader를 사용하여 수(num)을 입력받음
2. 전체 반복 for문
3.  num(입력받은 수)가 되기 전까지의 별과 빈칸을 채움
    if (i <= num) {
        for (빈칸입력)
        for (별 입력)
    }
    num이후의 별과 빈칸을 채움
    else
        for (빈칸)
        for (별)
